三、实验过程
1.输入文本，将文本转化为string格式。
2.将string改为stringbuffer计算出字符串长度。
3.建立循环每隔7个字加入“，”，14个字加入“。”。
4.接收输入的字符并统计出该字符出现的次数，若为空格或符号则报错。
